Freeway com JavaScript


1. Controles do jacaré:

Utilize as setas do teclado (cima, baixo, esquerda, direita) para movimentar o jacaré pela tela.


2. Objetivo:

O objetivo do jogo é fazer com que o jacaré atravesse a rua.


3. Pontuação:

Cada vez que o jacaré ultrapassar a linha de chegada, você ganhará 1 ponto.


4. Níveis do Jogo:

O jogo possui três níveis.

Para passar para o Nível 2, é necessário atingir 3 pontos.

Para passar para o Nível 3, é necessário atingir 2 pontos.


5. Ganhar o jogo:

Para vencer o jogo, é necessário atingir apenas 1 ponto no Nível 3.


6. Pontuação negativa:

Se a pontuação atingir -5 pontos, o jogador retorna ao nível anterior.


Jogar agora!
